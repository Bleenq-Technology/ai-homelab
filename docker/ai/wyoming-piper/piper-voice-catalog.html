Quick Reference — Wyoming Piper Installation

Steps to add any custom voice to wyoming-piper / Home Assistant

  1. Download both the .onnx and .onnx.json files (they must have matching base names).
  2. Copy them to your Piper data directory:
        • HA add-on: /share/piper/
        • Docker / wyoming-piper: /var/lib/wyoming-piper/data/ (or your mapped volume)
  3. Verify the .onnx.json has a "dataset" field that exactly matches the .onnx filename (without extension).
  4. For medium quality (~60 MB) set "quality": "medium"; for high (~110 MB) set "quality": "high". Sample rate is almost always 22050.
  5. Restart the Piper add-on / container, then restart Home Assistant.
  6. Navigate to Settings → Voice Assistants → your assistant → Text-to-speech and select the new voice from the dropdown.

💡 If the voice doesn't appear, the most common cause is a mismatch between the filename and the "dataset" value inside the JSON. Edit the JSON to match.
